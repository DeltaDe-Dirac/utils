# utils

A collection of utility scripts for common tasks.

---

## 📦 Available Scripts

### 🎵 yt2mp3

Convert YouTube videos to high-quality MP3 files with trimming support, metadata embedding, and intelligent caching.

**Quick Start:**
```bash
cd yt2mp3
./setup.sh              # Install dependencies
python yt_to_mp3.py "https://youtube.com/watch?v=VIDEO_ID"
```

📖 [Full Documentation →](yt2mp3/README.md)

## 🚀 Adding New Scripts

Each utility should:
- Live in its own subdirectory
- Include setup/install scripts
- Have comprehensive README
- Include unit tests
